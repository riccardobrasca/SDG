\chapter{The synthetic theory}

Let $R$ be a commutative ring.

\begin{definition}
  \leanok
  \lean{SDG.D}
We denote by $D$ the set
\[
\{r \in R \mbox{ such that } r ^ 2 = 0 \}
\]
\end{definition}
